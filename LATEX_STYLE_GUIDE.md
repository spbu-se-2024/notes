### Arrow-like symbols

|     Symbol      |           Command            |
|:---------------:|:----------------------------:|
|      `=>`       |            `\imp`            |
|      `<=`       |           `\limp`            |
|      `<=>`      |            `\iff`            |
| `=[sup/sub]=>`  |      `\ximp[sub]{sup}`       |
| `<=[sup/sub]=`  |      `\xlimp[sub]{sup}`      |
| `<=[sup/sub]=>` |      `\xiff[sub]{sup}`       |
|   `<=[def]=>`   |          `\iffdef`           |
|      `:=`       |         `\coloneqq`          |
|      `=:`       |         `\eqqcolon`          |
|      `->`       |        `\rightarrow`         |
|      `<-`       |         `\leftarrow`         |
|      `<->`      |      `\leftrightarrow`       |
|      `|->`      |          `\mapsto`           |
| `-[sup/sub]->`  |   `\xrightarrow[sub]{sup}`   |
| `<-[sup/sub]-`  |   `\xleftarrow[sub]{sup}`    |
| `<-[sup/sub]->` | `\xleftrightarrow[sub]{sup}` |
| `|-[sup/sub]->` |     `\xmapsto[sub]{sup}`     |


### Sets

|         Symbol         |  Command  |
|:----------------------:|:---------:|
|    Natural numbers     |   `\N`    |
| Natural numbers with 0 |   `\N*`   |
|        Integers        |   `\Z`    |
|    Rational numbers    |   `\Q`    |
|   Irrational numbers   |   `\I`    |
|      Real numbers      |   `\R`    |
|        Powerset        | `\2{set}` |


### Paired Delimiters

|  Symbol   |   Command    |
|:---------:|:------------:|
|  `(...)`  |  `\p{...}`   |
|  `{...}`  | `\set{...}`  |
|  `[...]`  | `\sqbr{...}` |
|  `<...>`  | `\geng{...}` |
|  `|...|`  | `\abs{...}`  |
| `||...||` | `\norm{...}` |
